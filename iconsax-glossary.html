Developer setup

Apps must register the Iconsax SVG interceptor before using @iconsax/... tokens.

  • Make sure every app that renders Iconsax icons provides this interceptor in the app module or root providers.
  • Use paths like @iconsax/outline/box, @iconsax/bold/calendar, or @iconsax/bulk/addCircle.
  • The interceptor maps those tokens to the self-hosted assets under assets/icons/iconsax.
tuiSvgInterceptor((src: TuiSafeHtml) =>
  String(src).startsWith('@iconsax/')
    ? `assets/icons/iconsax/${String(src).replace('@iconsax/', '')}.svg`
    : src
)
